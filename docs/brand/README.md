# Marca Poupito

**Conceito:** Crescimento Seguro. **Slogan:** *Descomplique, poupe, Poupito.*
**Tom:** amigável, acessível e encorajador — sem jargão financeiro.
**Público:** pessoas em geral querendo organizar a vida financeira de forma simples.

## Logo

- Símbolo: "P" azul-marinho com um broto verde crescendo (segurança + crescimento).
- Arquivos-fonte: `logo-lockup-fundo-branco.png` e `logo-lockup-transparente.png` (símbolo + wordmark "POUPITO · PERSONAL FINANCE").
- Ícones do app/PWA são recortados do símbolo — ver `web/public/icons/` e `web/public/favicon.ico`.
